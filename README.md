# M2yBecker


## Instalação

Adicionar ao Gemfile:

```ruby
gem 'm2y_becker'
```


## Configuração

Adicionar ao Gemfile:

```ruby

M2yBecker.configure do |config|
  config.main_url  = ""  
  config.username  = "cccc"
  config.password  = "xxx"

```
